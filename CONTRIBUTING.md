# Contributing to PayLeaf 🌿

Thanks for taking the time to contribute! Any help is welcome — bug fixes, features, design improvements, docs, anything.

## Getting Started

1. Fork the repo
2. Clone your fork
```bash
   git clone https://github.com/yourusername/payleaf.git
   cd payleaf
```
3. Install dependencies
```bash
   npm install
```
4. Copy env file and add your Supabase keys
```bash
   cp .env.example .env.local
```
5. Run locally
```bash
   npm run dev
```

## How to contribute

1. Create a branch
```bash
   git checkout -b feat/your-feature
```
2. Make your changes
3. Commit with a clear message
```bash
   git commit -m "feat: add currency selector"
```
4. Push and open a Pull Request

## Commit message format

- `feat:` new feature
- `fix:` bug fix
- `docs:` documentation
- `style:` styling only
- `refactor:` code restructure

## Good first issues

- Currency selector
- Google OAuth
- Better payslip PDF design
- CSV export
- More language support
